# bzplayer
----中文版说明----
B站所有视频倍速控制器脚本
调整B站视频播放速度
最高16倍速，最低0倍速
按+增加0.1倍速，按-下降0.1倍速

----English Version-----
station B all video speed controller script
adjust the video playback speed of station B
maximum 16 times speed, minimum 0 times speed
press + to increase speed by 0.1, press - to decrease speed by 0.1
